> [!note] Статус: стартовый ресёрч
> Конспект дополнительного анализа из связанной ChatGPT-сессии. Идеи ниже
> сохранены как входные гипотезы; решения и порядок работы находятся в
> [[../SEO-GROWTH-PLAN|каноническом SEO Growth Plan]].

**Дата:** 10 июля 2026 года  
**Источник:** дополнительный независимый разбор основного GEO+SEO-аудита  
**Назначение:** сохранить предложенные продуктовые, контентные и off-page идеи

## Основной тезис исходного разбора

Рост предлагалось строить в последовательности:

> достоверность → полезный продуктовый формат → оригинальные тесты → внешняя
> дистрибуция → техническая оптимизация.

Разбор справедливо предупреждал, что массовое производство новых статей или
покупка ссылок до исправления продуктовых фактов масштабирует недоверие.

## Предложенные направления и финальное решение

| Идея из разбора | Что в ней полезно | Решение для рабочего плана |
|---|---|---|
| Исправить factual/product contradictions | Непосредственный риск доверия и конверсии | **Принято, P0** |
| Создать единый Product Facts | Устраняет расхождения site/CWS/schema/content | **Принято, P0** |
| Бесплатный web prompt checker | Потенциально сильный product-led link magnet | **Только discovery, P2**: бесплатный аккаунт сейчас имеет 0 AI-операций; нужен отдельный cost/backend/rate-limit контракт |
| Страницы реальных интеграций | Высокая связь с продуктовым intent | **Принято, P1 после Product Facts**; только фактически поддерживаемые платформы |
| Task/use-case pages | Ближе к пользовательской работе, чем общие AI-темы | **Принято, P2 после SERP/query validation** |
| Broad guides + version pages | Разводит общий и version-specific intent | **Принято, P1**, первый кластер — Runway |
| Сравнения моделей | Коммерческий потенциал | **Условно принято**: только с собственными тестами и методикой |
| «Лаборатория промптов» и benchmark | Создаёт первичные данные, ссылки и видео | **Принято, P1/P2** |
| YouTube/VC/Habr/Reddit/partners | Усиливает дистрибуцию и brand entity | **Принято, P1**, без спама и поддельной независимости |
| Покупные dofollow/PBN/накрутка | Краткосрочная манипуляция | **Отклонено** |
| Платные нативные публикации | Могут дать охват и брендовые упоминания | **Допустимо** с disclosure и `rel="sponsored"`/`nofollow` |
| Массовые model/doorway pages | Даёт формальный охват | **Отклонено** без самостоятельной ценности каждой страницы |
| Публичный текстовый слой Learn | Улучшает открытые уроки | **Принято только для public Learn**; платный Kinescope-курс остаётся закрытым/noindex |
| Bing Webmaster Tools | Даёт Bing/index/AI-citation baseline | **Принято, P1** |
| `llms.txt`/schema как главный рычаг | Легко реализовать | **Понижено**: точность, intent и authority важнее формального GEO score |

## Идеи первых linkable assets

1. Runway vs Kling vs PixVerse: одинаковая сцена и фиксированная методика.
2. Compatibility matrix для negative prompt по моделям и платформам.
3. Стоимость одного пригодного рекламного ролика в нескольких генераторах.
4. Prompt на русском против prompt на английском.
5. Влияние длины и структуры prompt на процент удачных генераций.
6. Проверка популярных советов по промптингу на воспроизводимость.

Каждый эксперимент предлагалось раскладывать на исходное исследование,
YouTube, короткие видео, Telegram, внешнюю публикацию и партнёрский outreach.

## Ограничения этого документа

- Это конспект предложения, а не утверждённая спецификация.
- Числа, позиции и возможности внешних платформ меняются; перед внедрением
  используются свежие данные GSC/Яндекс/Bing и официальные product docs.
- Любое изменение auth, billing, course access, extension integration или
  Supabase contract подчиняется `AGENTS.md` и `docs/INTEGRATION-CONTRACT.md`.
