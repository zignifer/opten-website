- `ogTitle: string` — `<meta property="og:title">`
- `ogDescription: string` — `<meta property="og:description">`
- `ogImage?: string` — optional absolute URL; when omitted the prerender script falls back to DEFAULT_OG_IMAGE
- `prerender: "full" | "head" | "none"` — D-02 tier (full = head + body, head = head only, none = skip)
- `changefreq: "weekly" | "monthly" | "yearly"`
- `priority: number` — 0..1; matches Phase 1 `public/sitemap.xml` priorities (1.0, 0.8, 0.6, 0.3, 0.3, 0.3)
