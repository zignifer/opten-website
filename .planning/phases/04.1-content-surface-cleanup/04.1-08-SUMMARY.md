# Phase 4.1 Plan 08: prerender.mjs SSR body length assert (IN-03) — Summary

One-liner: Defense-in-depth build guard — prerender.mjs now throws when a full-tier RouteMeta has no matching `<Route>` in entry-server.tsx, so the silent "head-only HTML body" regression mode is impossible to ship.

## What landed

One atomic commit `e539a1c`, single file modified:

- **`scripts/prerender.mjs`** — Inside the existing full-tier branch of the route loop, after `renderRoute(meta.path)` returns and before `applyBody(...)` is called, added a 200-char length check that throws with a clear error message naming the offending path. 7 lines added (5-line comment + 2-line if/throw block); zero changes to `applyBody`, `applyMarker`, `applyPaddleScript`, the loop structure, or any other apply* helper.
