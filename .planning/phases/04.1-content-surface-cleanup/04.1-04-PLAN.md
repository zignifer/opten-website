<objective>
Close WR-08 (`readonly FaqItem[]` variance) with one atomic commit touching two files. Property typing tightening — accepts MORE values (readonly arrays), rejects nothing that previously compiled.

Purpose: `src/content/landingFaq.ts` exports `landingFaq = { ru, en } as const`, which makes `landingFaq.ru` a `readonly FaqItem[]`. The current `FaqBlock` `items: FaqItem[]` prop and the `faqPageBlock(items: { q: string; a: string }[], pageId)` signature both want a mutable array. Pre-emptive fix: widen both signatures to `readonly ...[]`.
