## Suggested Placement

- Title/H1: natural phrase around `создать сайт с нейросетью` or `нейросеть для создания сайтов`.
- Intro: clarify that AI can make a draft, but strategy, content, UX and QA still matter.
- H2 ideas: `Что подготовить до генерации`, `Prompt для лендинга`, `Что проверять после AI`, `Как превратить это в услугу`.
- FAQ: `Можно ли создать сайт с нейросетью бесплатно?`, `Какие данные нужны для prompt?`, `Можно ли брать заказы?`

## Article Angle

This is the practical follow-up to vibe coding. Focus on one deliverable: landing page. Show the full path, not just the "AI generated code" step.

## Outline Suggestions

1. Что AI умеет в создании сайтов.

3. Prompt for landing page.
4. Review checklist: mobile, CTA, copy, speed, forms.
5. How this can become first freelance service.
6. Opten prompt preflight.

## Practical Case

Новичок хочет сделать сайт для локального бизнеса. Vague prompt gives a generic template. A detailed prompt with offer, audience, sections, proof and visual style gives a useful landing draft.
