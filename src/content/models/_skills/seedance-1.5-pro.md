# Seedance 1.5 Pro — Skill для PromptScore

> Этот документ используется AI-валидатором PromptScore как база знаний при оценке промптов для модели Seedance 1.5 Pro.
> Seedance 1.5 Pro — промежуточная версия между 1.0 и 2.0, с улучшенным качеством движений и расширенными возможностями камеры.

## Отличия от Seedance 2.0

Seedance 1.5 Pro — значительное улучшение над 1.0, приближающееся к возможностям 2.0, но с некоторыми ограничениями:

| Возможность | Seedance 2.0 | Seedance 1.5 Pro |
|-------------|-------------|-----------------|
| Разрешение | До 2K | До 1080p |
| Длительность | 4-15 сек | 5 или 10 сек |
| Мультимодальный вход | Изображения + видео + аудио (до 12 файлов) | Текст + 1-2 изображения |
| @-референсы (полные) | До 9 изображений + 3 видео + 3 аудио | Ограниченная поддержка (1-2 изображения) |
| Consistency Control | Полный | Базовый (через referece-изображение) |
| Звуковой контроль | Да | Нет |
| Негативные промпты | Ограниченно | Базовая поддержка (лучше чем 1.0, хуже чем 2.0) |
| Тайминг-раскадровка | Полная (по секундам) | Ограниченная (начало/середина/конец) |
| Камерные движения | Полный набор + спецприёмы | Расширенный набор (больше чем 1.0) |
| Качество движений | Отличное | Хорошее (заметно лучше 1.0) |
| Сложные сцены | Высокая стабильность | Средняя стабильность |

**Отличия от 1.0 Pro:** Улучшенная физика движений, более стабильная генерация сложных сцен, лучшее следование промптам, базовая поддержка референс-изображений, ограниченная поддержка негативных промптов.

## Структура хорошего промпта

Оптимальная структура промпта для Seedance 1.5 Pro:

```
[Subject/Character] + [Action/Motion] + [Scene/Environment] + [Camera Movement] + [Style/Mood]
```

По сравнению с 1.0 Pro, можно писать более детализированные промпты — модель лучше их обрабатывает.

### Формула Text-to-Video
```
Субъект + Движение + Сцена + Камера + Стиль/Настроение
```

### Формула Image-to-Video
```
Субъект + Движение, Фон + Движение, Камера + Движение + Стиль
```

### Простая раскадровка (поддерживается в 1.5)
```
Начало: [описание]
Середина: [описание]
Конец: [описание]
```
Не такая точная как тайминг по секундам в 2.0, но даёт базовый контроль над нарративом.

## Поддерживаемые типы генерации

### 1. Чистый текст (Text-to-Video)
Генерация из текстового описания.
- Формула: `(Субъект) + (Действия) + (Среда/Свет) + (Камера) + (Стиль)`
- Пример: «Cinematic shot of a samurai standing in a bamboo forest at dawn. Wind moves through the bamboo. He slowly draws his sword, camera orbits from behind to front. Golden hour light, film grain, 1080p, 10s»

### 2. Изображение в видео (Image-to-Video)
Анимация статичного изображения с детальным контролем.
- Формула: `[Субъект + движение] + [Фон + движение] + [Камера + движение]`
- Улучшенное следование деталям изображения по сравнению с 1.0

### 3. Базовый Reference Control
Ограниченная поддержка reference-изображений для сохранения стиля.
- Возможно использовать 1-2 изображения как визуальный ориентир
- Не такой точный как Consistency Control в 2.0
- Формула: `[Описание сцены] + reference image для стиля/персонажа`

### НЕ поддерживается в 1.5 Pro:
- Полный Consistency Control с @-референсами (множественные изображения)
- Копирование камеры/движений из видео-референса
- Продление видео (Video Extension)
- Звуковой контроль и клонирование голоса
- One-Take Long Shot с несколькими @Image референсами
- Редактирование существующего видео
- Музыкальная синхронизация (Beat Sync)
- Точная тайминг-раскадровка по секундам

## Камерные движения

Seedance 1.5 Pro имеет расширенную поддержку камерных движений — значительно лучше 1.0.

### Словарь камерных движений

| Категория | Ключевые слова |
|-----------|---------------|
| Крупность | Дальний план, общий план, средний план, крупный план, сверхкрупный план, макро |
| Движение | Наезд, отъезд, панорама, трекинг, орбита, crane shot, dolly, push in, pull out |
| Угол | Нижний ракурс, верхний ракурс, птичий взгляд, голландский угол, POV |
| Ритм | Слоу-моушн, ускоренная съёмка |
| Фокус | Малая глубина резкости, перевод фокуса, размытие фона |
| Спецприёмы | Camera switching, cut to, wrap around, follow shot |

### Важные правила камеры
- **Non-fixed camera** — обязательно при движениях камеры
- **Fixed camera** — для статичных сцен
- Поддерживает сложные комбинации (orbit + zoom), но не более 2 одновременно
- Camera switching работает стабильнее чем в 1.0

## Продвинутые техники (доступные в 1.5 Pro)

### Степень выраженности через наречия
Как и в 1.0, но модель лучше интерпретирует нюансы:
- Полезные наречия: fast, intense, large, high frequency, strong, crazy, gently, slowly, gradually

### Множественные действия
1.5 Pro стабильнее обрабатывает цепочки действий:
- Поддерживает 3-5 последовательных действий в 10-секундном видео
- Camera switching между действиями работает надёжнее

### Простая раскадровка
Можно указать что происходит в начале, середине и конце:
```
Start: woman sits at a café table, reading a book. Medium shot.
Middle: she looks up, sees someone, smiles. Camera slowly zooms in.
End: close-up of her face, warm smile, golden hour light.
```

### Базовые негативные промпты
В отличие от 1.0, простые запреты могут работать:
- «No text, no watermarks» — может сработать
- Сложные негативные конструкции — ненадёжно
- Рекомендация: всё равно лучше писать **позитивно**

## Критерии оценки промпта (для AI-валидатора)

При оценке промпта AI должен учитывать промежуточный статус 1.5 Pro:

### Обязательные элементы (без них генерация будет слабой)
1. **Субъект** — кто или что в кадре. Конкретно, не абстрактно.
2. **Действие** — физические действия, конкретные глаголы. «Идёт, поворачивается, берёт» вместо «что-то происходит».
3. **Камера** — тип камеры (fixed/non-fixed), крупность плана или движение.
4. **Сцена/среда** — где происходит действие (для Text-to-Video).

### Бонусные элементы (улучшают результат)
5. **Стиль/настроение** — визуальный стиль, цветовая палитра, атмосфера.
6. **Освещение** — конкретные источники света и их характер.
7. **Простая раскадровка** — начало/середина/конец для 10-секундных видео.
8. **Наречия степени** — для точного контроля интенсивности.
9. **Детали персонажа** — внешность, одежда, эмоции.
10. **Базовые параметры** — разрешение, соотношение сторон, длительность.
11. **Язык** — английский даёт лучшие результаты на syntx.ai.

### Антипаттерны (ухудшают результат)
- Слишком короткий промпт (<10 слов) — модель будет додумывать слишком много
- Слишком длинный промпт (>1500 символов) — перегрузка деталями
- **Сложные негативные промпты** — базовые («no text») могут работать, сложные — нет
- **Конфликт с изображением** — описание не совпадает с загруженным изображением
- Полный @-синтаксис (множественные @Image/@Video) — не поддерживается как в 2.0
- Точная тайминг-раскадровка по секундам — не поддерживается (использовать начало/середина/конец)
- Запрос на >10 секунд — невозможно
- Более 2 одновременных камерных движений — нестабильно
- Абстрактные/размытые формулировки
- Запрос звука/диалогов — не поддерживается

### Специфичные для 1.5 Pro рекомендации
- Поощрять использование простой раскадровки (начало/середина/конец) — это сильная сторона 1.5
- Поощрять детализированные описания — модель хорошо их обрабатывает
- Если пользователь пишет тайминг по секундам — предложить упростить до начало/середина/конец
- Если используется полный @-синтаксис 2.0 — предупредить об ограничениях
- Промпт может быть сложнее чем для 1.0, но проще чем для 2.0 — это нормально
- Camera switching + orbit/follow — сильная комбинация для 1.5 Pro

### Контекстная оценка
AI должен понимать НАМЕРЕНИЕ пользователя:
- 1.5 Pro — модель для **качественных коротких видео с хорошим контролем камеры**
- Промежуточная по возможностям — ожидать промпты среднего уровня сложности
- Не наказывать за отсутствие техник 2.0 (полные @-референсы, звук, Beat Sync)
- Поощрять использование расширенных камерных движений — это сильная сторона
- Оценивать соответствие промпта возможностям ЭТОЙ модели
