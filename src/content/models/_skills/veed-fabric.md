**КРИТИЧЕСКИ ВАЖНО:** Fabric — это **НЕ генератор видео общего назначения**. Это специализированная модель **lip-sync / talking head**. Вход: изображение + аудио. Текстовый промпт в традиционном понимании НЕ используется.

## Как работает Fabric

### Основной режим: Image + Audio
1. Загружаете изображение (любой стиль: фото, иллюстрация, аниме, 3D, глина, бренд-маскот)
2. Загружаете аудио (речь, музыка)
3. Модель генерирует видео с lip-sync, движениями головы, тела и жестами рук

### Text-to-Speech режим (через VEED)
Вместо аудио пишется **речевой скрипт** — текст, который TTS (ElevenLabs V3) озвучивает.

## Fabric Emotions — inline теги

При использовании TTS-режима, в скрипт встраиваются emotion-теги `[tag]`:

### Эмоции
`[excited]`, `[happy]`, `[sad]`, `[angry]`, `[curious]`, `[nervous]`, `[confident]`

### Реакции
`[laughs]`, `[sighs]`, `[gasps]`, `[clears throat]`

### Громкость
`[whispers]`, `[shouting]`

### Темп
`[pause]`, `[long pause]`, `[breathes]`, `[rushed]`, `[drawn out]`

### Звуковые эффекты
`[applause]`, `[gunshot]`, `[door creaks]`

### Акцент
`[American accent]`, `[British accent]`

**Правило:** Не перегружать теги — распределять постепенно для естественного результата.

## Критерии оценки промпта (для AI-валидатора)

### Особый статус Fabric

Fabric — **НЕ text-to-video модель**. Оценка «промпта» здесь означает оценку **речевого скрипта** (если используется TTS-режим) или просто проверку наличия входных данных.

### Для TTS-режима (речевой скрипт)

#### Обязательные элементы
1. **Текст речи** — разборчивый, осмысленный текст для озвучивания.
2. **Входное изображение** — если нет — fail, score не выше 20.

#### Бонусные элементы
3. **Emotion теги** — `[excited]`, `[happy]`, `[sad]` и т.д. для эмоциональной экспрессии.
4. **Темповые теги** — `[pause]`, `[rushed]`, `[drawn out]` для контроля ритма.
5. **Акцент** — `[American accent]`, `[British accent]` при необходимости.
6. **Постепенное распределение тегов** — не перегружать.

### Для Audio-режима
- Текстовый промпт НЕ используется. Качество зависит от аудио и изображения.
- Чёткое аудио с минимальным фоновым шумом.
- Фронтальное, хорошо освещённое изображение с видимым лицом.

### Антипаттерны (ухудшают результат)
- **Промпт с описанием сцены** — Fabric НЕ генерирует сцены, он анимирует лица.
- **Ожидание кинематографических сцен** — Fabric = lip-sync only.
- **Экстремальные ракурсы на входном изображении** — сильные углы и окклюзии ухудшают результат.
- **Перегрузка emotion тегами** — не ставить тег на каждое слово.
- **Грязный аудио** — фоновый шум деградирует lip-sync.
- **30-секундный лимит** в Studio — для длинного контента использовать API.
- **Камерные движения и экшн** — модель НЕ поддерживает.

### Контекстная оценка
AI должен понимать:
- Fabric принципиально отличается от T2V моделей
- «Промпт» = речевой скрипт для TTS, не описание видеосцены
- Emotion теги `[tag]` — фича модели, не ошибка форматирования
- Короткий скрипт — нормально, если контент осмысленный
- Качество определяется парой image + audio, не текстом
- 30+ языков поддерживается — любой язык аудио нормален
