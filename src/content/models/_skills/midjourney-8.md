## Ключевое отличие V8 от V7

**V8 — это НЕ улучшенная V7. Это другая модель с другим подходом.**

### Что изменилось
1. **Более буквальная интерпретация** — V7 творчески додумывала то, что вы не указали. V8 генерирует ровно то, что описано — не больше.
2. **Реалистичная эстетика по умолчанию** — V7 автоматически добавляла художественную обработку. V8 даёт нейтральный, фотографический результат. Для стилизации нужно явно просить.
3. **Улучшенный рендеринг текста** — 1–3 слова (вывески, упаковка, надписи) теперь надёжно генерируются. Оборачивайте текст в кавычки.
4. **Нативное 2K разрешение** — через `--hd` генерирует 2048x2048 без апскейлера. Каждый пиксель генерируется с нуля.
5. **Скорость ~5x** — генерация примерно в 5 раз быстрее, чем V7.
6. **Персонализация стала центральной** — Style Creator, персональные коды, мудборды — не опции, а основной рабочий процесс.

### Практические последствия
- Промпт «woman in a forest» в V7 давал что-то красивое с тёплым светом. В V8 — технически точное, но плоское изображение.
- Нужно ЯВНО указывать освещение, настроение, композицию, стиль.
- Спецификация > интерпретация.

## Структура хорошего промпта для V8

V8 вознаграждает связные описания, а не списки ключевых слов.

### Главные правила

1. **Указывайте освещение ЯВНО** — это самое важное изменение. V8 не добавит интересный свет автоматически.
2. **Пишите связными описаниями** — не «mountain, fog, sunrise, epic» а «A snow-capped mountain emerging from fog at sunrise, dramatic warm light on the peaks».
3. **Добавляйте камеру и оптику** — V8 отлично реагирует на терминологию фотографии.
4. **Используйте --no осознанно** — V8 значительно лучше исполняет негативные промпты.
5. **Фиксируйте --ar с самого начала** — смена соотношения сторон = начало с нуля.

### Шаблон промпта

```
[СУБЪЕКТ с деталями], [СРЕДА/КОНТЕКСТ], [КОНКРЕТНОЕ ОСВЕЩЕНИЕ], [КАМЕРА/ОПТИКА], [СТИЛЬ/НАСТРОЕНИЕ] --ar X:Y --s N --v 8
```

### Примеры освещения для V8

Вместо: `dramatic portrait, studio lighting`
Пишите: `Portrait lit from a single softbox positioned 45 degrees to the left, creating a strong shadow across the right side of the face, against a dark gray background`

Эффективные описания:
- `soft overcast natural light`
- `warm backlight creating rim lighting on the hair`
- `harsh midday sun from above`
- `neon reflections on wet pavement at night`
- `golden hour directional light from the right, casting long shadows`

## Параметры V8

### Основные параметры

| Параметр | Диапазон | По умолчанию | Описание |
|----------|----------|-------------|----------|
| `--v 8` | — | — | Указывает версию V8 |
| `--ar` | Любое соотношение | 1:1 | Соотношение сторон |
| `--hd` | — | выкл | Нативное 2K разрешение (2048x2048). Стоимость 4x |
| `--q 4` | — | 1 | Усиленная когерентность. Стоимость 4x |
| `--s` / `--stylize` | 0–1000 | 100 | Художественная интерпретация |
| `--c` / `--chaos` | 0–100 | 0 | Разнообразие вариаций |
| `--w` / `--weird` | 0–3000 | 0 | Экспериментальность |
| `--no` | текст | — | Негативный промпт (работает значительно лучше, чем в V7) |
| `--style raw` | — | выкл | Убирает гиперполированный дефолтный стиль V8 |
| `--exp` | 0–100 | 0 | Экспериментальная детализация |
| `--p` | код или без | — | Персонализация |
| `--sref [URL]` | — | — | Style Reference (стоимость 4x в alpha) |
| `--sw` | 0–1000 | 100 | Вес style reference |
| `--seed` | 0–4294967295 | случайный | Воспроизводимость |

### Стилизация в V8: рекомендуемые диапазоны

| Диапазон --s | Эффект | Подходит для |
|-------------|--------|-------------|
| 0–50 | Максимально буквально | Продуктовые фото, техреференсы |
| 50–150 | Сбалансированный | Фотореалистичные работы (РЕКОМЕНДУЕТСЯ) |
| 200–400 | Художественные решения | Редакционные, маркетинговые работы |
| 500–800 | Свободная интерпретация | Может отклоняться от промпта |
| 1000 | Максимальная интерпретация | Промпт = свободное вдохновение |

**Важно:** В V8 рабочий диапазон --s 100–400. Экстремальные значения (0 и 1000) дают менее радикальную разницу, чем в V7.

### Комбинация --style raw + низкий --s

Для максимального контроля:
```
--s 50 --style raw --v 8
```
Отключает и художественную интерпретацию (--s), и дефолтную обработку (--style raw). Идеально для продуктовых фото и техиллюстраций.

## Draft Mode

V8 включает режим черновика для быстрой итерации:
- Генерирует значительно быстрее при пониженном качестве
- Используйте для проверки направления промпта
- Переходите на полное качество только когда промпт даёт нужный результат

## Персонализация и Style Creator

### Персонализация (--p)
- Строится на основе рейтинга изображений (thumbs up / down)
- Качество рейтинга важнее количества — оценивайте осознанно
- Можно делиться кодами: `--p abc123`
- Усиливается высоким --s: `--s 300 --p` = сильный личный стиль
- Ослабляется низким --s: `--s 100 --p` = мягкая коррекция

### Style Creator
1. Откройте инструмент в веб-интерфейсе MidJourney
2. Выберите из сгенерированных вариаций те, что соответствуют целевому стилю
3. Получите многоразовый код стиля
4. Применяйте через `--style [ваш-код]`

**Стратегия:** Определите стиль в начале проекта через Style Creator, получите код, применяйте его ко всем промптам проекта.

### Style Reference (--sref)
- Переносит визуальный стиль из изображения (палитра, текстура, освещение, композиция)
- `--sw` контролирует силу (выше = ближе к стилю)
- Тестируйте при --sw 50, 100, 200 для калибровки
- Можно комбинировать: `--sref [URL] --p [код]` — стилевой референс + персонализация

### Мудборды
- Новая версия --sv 7: в 4 раза быстрее и дешевле
- Поддерживают --hd, --p, --stylize, --exp
- Можно смешивать с --sref кодами в одном промпте

## Известные проблемы V8 Alpha

| Проблема | Описание | Решение |
|----------|----------|---------|
| Гиперполированность | Дефолтные выдачи выглядят слишком обработанными, искусственными | `--style raw` |
| Ограниченная абстракция | Модель «исправляет» сюрреалистические промпты в что-то реалистичное | Высокий --weird, --chaos |
| Возрастной дрейф | Субъекты выглядят старше указанного | Явно указывать возраст + «youthful features» |
| «Minecraft-эффект» | Иногда блочные текстуры | Перегенерация, изменение формулировок |
| Нестабильность | 3 из 4 результатов отличные, 1 — промах | Нормально для Alpha |

A middle-aged man with weathered skin and kind eyes, wearing a worn denim jacket, standing in a doorway of an old bookshop, soft afternoon light from a nearby window casting warm shadows across his face, shot on an 85mm lens f/1.8, shallow depth of field, editorial photography --ar 4:5 --s 100 --v 8

## Критерии оценки промпта (для AI-валидатора)

При оценке промпта AI должен учитывать:

### Обязательные элементы (без них V8 даст плоский результат)
1. **Субъект** — конкретный, в начале промпта. V8 весит первые слова тяжелее всего.
2. **Освещение** — КРИТИЧНО для V8. Без явного указания результат будет технически точным, но визуально плоским и скучным.
3. **Контекст/Среда** — где находится субъект, что вокруг.

### Бонусные элементы (V8 вознаграждает их больше, чем V7)
4. **Камера/Оптика** — V8 отлично реагирует на фотографические термины: «85mm f/1.8», «medium format».
5. **Стиль/Настроение** — V8 НЕ добавляет настроение автоматически. Если не указано — результат нейтральный.
6. **Цветовая палитра** — конкретные цвета или направление: «warm autumn tones», «teal and orange color grade».
7. **Персонализация/Sref** — использование --p, --sref, Style Creator для консистентного стиля.
8. **Текст в кавычках** — если нужен текст в изображении, оборачивайте в "кавычки".
9. **Осмысленный --ar** — V8 адаптирует композицию под соотношение сторон. Решайте его в начале.

### Антипаттерны (ухудшают результат в V8)
- **Промпт в стиле V7** — списки ключевых слов через запятую вместо связных описаний дают плоские результаты
- **Отсутствие освещения** — самая частая ошибка. V8 не подставит «красивый свет» за вас
- **Спам качества** — «beautiful, stunning, 8k, detailed, masterpiece» — V8 это полностью игнорирует
- **Оставить --s по умолчанию для всего** — тестируйте --s 50, 200, 500 для каждого типа контента
- **Персонализация без рейтинга** — --p бесполезен, если вы оценили мало изображений
- **Нагромождение модификаторов** — --sref + --p + --style + высокий --s + 10 стилевых слов = хаос
- **Пропуск Draft mode** — полноценные рендеры V8 дорогие (особенно --hd). Итерируйте в Draft
- **Промпт «mountain, fog, sunrise, epic»** — V8 даст буквально гору, туман и восход, без художественности
- **Абстрактные формулировки** — «something interesting» не работает в V8 ещё хуже, чем в V7
- **--hd + --q 4 одновременно** — работает только в Fast режиме, не в Relax. Стоимость 4x

### Контекстная оценка
AI должен понимать НАМЕРЕНИЕ пользователя:
- V8 Alpha — экспериментальная модель, поведение может меняться
- Для фотореализма: --s 50–150 + --style raw = лучший диапазон
- Для иллюстраций: --s 300–600 добавляет визуальный интерес
- Если промпт простой и без освещения — это главная рекомендация для улучшения
- Текст в изображении — сильная сторона V8, стоит поощрять использование кавычек
- Нативное 2K (--hd) стоит 4x — рекомендуйте только для финальных рендеров
- Если пользователь пришёл из V7, главный совет: «добавьте освещение и пишите описаниями»
