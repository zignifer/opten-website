## Структура хорошего промпта

Подробные описательные промпты с композицией сцены. Поддерживает билингвальные промпты (EN + CN).

```
[Main subject] + [Scene composition] + [Style] + [Text content to render] + [Layout details]
```

### Принципы

- Указывать текст для рендера явно в промпте
- Стилевые и композиционные ключевые слова поддерживаются
- V2.0 поддерживает до 1,000 токенов — можно описывать сложные композиции (инфографика, комиксы)
- Для editing: описание инструкции редактирования

### Типографика (ключевая фича)

Qwen Image — лидер в рендеринге текста:
- Multi-line layouts
- Paragraph-level text
- Инфографика с графиками и текстовыми блоками
- PPT слайды
- Комиксы с текстовыми пузырями
- Постеры с заголовками

## Критерии оценки промпта (для AI-валидатора)

При оценке промпта AI должен учитывать:

### Обязательные элементы (без них генерация будет слабой)
1. **Субъект/контент** — что изображено или какой документ создаётся.
2. **Стиль** — визуальный стиль: фотореализм, иллюстрация, корпоративный и т.д.
3. **Текст для рендера** (если нужен) — явное указание текста в промпте.

### Бонусные элементы (улучшают результат)
4. **Композиция/layout** — для сложных документов (инфографика, комиксы).
5. **Типографика** — тип шрифта, размер заголовков, расположение текста.
6. **Цветовая схема** — для корпоративных и маркетинговых материалов.
7. **ControlNet** — использование структурного контроля.
8. **Язык** — EN и CN оба нативны, не штрафовать за китайский.

### Антипаттерны (ухудшают результат)
- **Путаница с Qwen2.5-VL** — Qwen Image = генератор. Qwen2.5-VL = анализатор/vision. Разные модели.
- **Путаница с Z-Image** — другая команда Alibaba, другая модель.
- **Промпт без явного текста** для документов — если нужен текст в изображении, указать его.
- **V1 на слабом GPU** — V1 требует 40–58GB VRAM.

### Контекстная оценка
AI должен понимать:
- Qwen Image ≠ Qwen2.5-VL (vision), ≠ Z-Image (другая команда)
- Типографика — ключевая фича, промпты для текстового контента оценивать по качеству текстовых инструкций
- 1,000 токенов — можно и НУЖНО использовать для сложных композиций (инфографика, PPT)
- Билингвальный контент — EN + CN нативны
- Editing промпты — как инструкции, не полное описание
- Для простых изображений без текста — стандартная оценка субъект + стиль + описание
